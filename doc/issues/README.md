# KitchenAI — Issue Index

This directory contains individual issue files split from the main [feature specification](../feature-specifications.md).
Each file corresponds to a user story or technical task that can be tracked as a GitHub issue.
Issues that contain both frontend and backend work are further split into `-BE` (backend) and `-FE` (frontend) sub-issues.
